# Sales & Revenue Analysis Dashboard

## Objective
Build a dashboard to analyze sales and revenue data using Power BI.

## Tools Used
- Power BI
- Excel

## Features
- Total Revenue KPI
- Total Sales KPI
- Product Count KPI
- Revenue Trend Analysis
- Top Performing Products
- Category-wise Revenue Analysis
- Interactive Filters and Slicers

## Insights
- Laptop generates the highest revenue.
- Electronics contribute the majority of revenue.
- Revenue varies across different days.
- Product-wise analysis helps identify profitable products.
